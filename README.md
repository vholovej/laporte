# La Porte Configurator

MVP внутрішнього веб-додатку для салону La Porte: менеджер створює комерційну пропозицію, додає дверні позиції, бачить автоматичний розрахунок і формує PDF.

## Що є в MVP

- список КП зі статусами, пошуком, датою, менеджером і сумою;
- створення, редагування, дублювання та видалення КП;
- форма клієнта;
- конфігуратор багатьох дверних позицій;
- умовна база цін у `src/data/pricing.json`;
- автоматичний розрахунок позиції, знижки та фінальної суми;
- проста 2D-схема дверей;
- PDF-комерційна пропозиція українською;
- збереження в `localStorage`;
- три тестові КП після першого запуску.

## Як запустити локально

Потрібен Node.js 20+.

```bash
npm install
npm run dev
```

Після запуску відкрийте адресу, яку покаже термінал. Зазвичай це:

```text
http://localhost:5173
```

## Де змінювати ціни

Усі базові ціни, націнки, фурнітура та послуги лежать у:

```text
src/data/pricing.json
```

Після зміни файлу перезапустіть dev-сервер або дочекайтеся автоматичного оновлення Vite.
